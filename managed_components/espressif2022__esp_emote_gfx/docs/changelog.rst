Changelog
=========

Version 1.0.0
-------------

Initial release of ESP Emote GFX framework.

Features:

* Core graphics system with buffer management
* Object system for all graphical elements
* Image widget (RGB565, RGB565A8)
* Label widget with LVGL and FreeType font support
* Animation widget (EAF format)
* QR code widget
* Timer system for animations
* Thread-safe operations
